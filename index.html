GitHub арқылы сайт жасау және Wix сайтым туралы мәліметтер

Портфолио

Төменде GitHub арқылы HTML сайт жасауға арналған толық қадамдар берілген. Бұл процесс веб-бағдарламалау саласында алғашқы қадам жасауға көмектеседі.

GitHub арқылы сайт жасау алгоритмі:

  1. GitHub аккаунтын жасаңыз: github.com сайтына кіріп, тіркеліңіз. Тіркелу процесін аяқтаған соң, электрондық поштаңызды растаңыз.
  2. Жаңа репозиторий құрыңыз: "New Repository" батырмасын басыңыз. Репозиторийге ат қойыңыз (мысалы, "portfolio-site").
  3. HTML және CSS файлдарын дайындаңыз: Сайттың негізгі құрылымын жасаңыз:
    • index.html – бұл басты бет;
    • style.css – бет дизайнын басқаратын файл.
  4. Файлдарды жүктеңіз: Репозиторийге HTML, CSS, JavaScript файлдарыңызды жүктеп, "Commit changes" батырмасын басыңыз.
  5. GitHub Pages-ті іске қосыңыз: Репозиторийдің "Settings" → "Pages" бөліміне кіріп, "main" тармағын таңдаңыз. GitHub сіздің сайтыңызды жариялайды.
  6. URL мекенжайын алыңыз: GitHub сізге сайттың URL мекенжайын береді (мысалы, https://username.github.io/portfolio-site).

Менің HTML кодым:

<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>Welcome to My Portfolio</h1>
    <p>This is a simple HTML page.</p>
</body>
</html>
